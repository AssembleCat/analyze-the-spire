# Slay the Spire 분석기 (Slay the Spire Analyzer)

슬레이 더 스파이어(Slay the Spire) 게임 데이터를 수집, 처리 및 분석하는 프로젝트입니다. 게임 플레이 로그를 크롤링하여 전투 데이터를 추출하고, 딥러닝 모델을 통해 게임 내 의사결정을 분석합니다.
